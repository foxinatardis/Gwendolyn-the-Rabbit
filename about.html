How:

Utilizing a JavaScript library called Phaser for game creation and manipulation. It takes adavantage of HTML5 standards.

An Express.js server runs the back end, it is deployed via Heroku.

Currently it is recommended to play "Gwendolyn the Rabbit" in Chrome or Safari. Additional browser support slated for future development.
